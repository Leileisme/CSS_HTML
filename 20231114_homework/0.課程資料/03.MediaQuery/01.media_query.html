響應式網頁設計(Responsive web design, RWD)

  • 1.響應式網頁設計，讓內容如流水般能適應各種形狀的容器，能在每個裝置尺寸正常瀏覽。
  • 2.css的媒體查詢(Media Query)負責想樣式最主要的工作
  • 3.媒體查詢的斷點(breakpoint)為不同裝置尺寸設定通過的位置點，當通過設定的位置套用該尺寸的樣式。
  • 4.行動裝置優先的漸進增強的概念，基於小尺寸開始思考內容的設計模式。
  • 5.使用相對單位設定寬、高或大小(rem、%、vh、vw)，字體大小使用 rem，容器寬度使用 %
