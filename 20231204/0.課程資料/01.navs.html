JavaScript 行為

使用 data 屬性設定

  • 先確認使用 .nav-tabs 或 .nav-pills
  • 如果要呈現分頁頁籤加上 data-bs-toggle="tab" 啟動 tab 功能
  • 如果要呈現膠囊狀加上 data-bs-toggle="pill"，啟動 pill 功能
  • data-bs-target 的值要對應內容呈現區的 id，例如 #home 對應 id="home"
